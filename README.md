# ripgrim

Personal AI skills repository for OpenClaw agents.

## 📦 Skills

This repository contains custom skills for AI agents to extend their capabilities with specialized knowledge, workflows, and tool integrations.

[View all skills →](.skills/)

## 🚀 Quick Start

### Install a Skill

```bash
npx skills add ripgrim/ripgrim@skill-name -g
```

### Available Skills

- **ai-sdk** - AI SDK integration and best practices
- **frontend-design** - Frontend UI/UX design patterns
- **typescript** - TypeScript best practices and patterns
- **react** - React development guidelines
- **convex** - Convex backend development
- And [many more →](.skills/)
